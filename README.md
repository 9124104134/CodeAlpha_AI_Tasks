# CodeAlpha_AI_Tasks

This repository contains the completed Artificial Intelligence internship tasks assigned by CodeAlpha. The project showcases real-world implementations of Natural Language Processing (NLP), text similarity matching, and Computer Vision (CV) workflows.

## 🚀 Projects Overview

### Task 1: Language Translation Tool
* **Description:** A user-friendly web application that allows users to input text and translate it into multiple target languages instantly.
* **Tech Stack:** Python, Streamlit, Deep-Translator API.

### Task 2: Chatbot for FAQs
* **Description:** An intelligent conversational agent that matches user queries to an internal FAQ database using advanced NLP preprocessing and Cosine Similarity evaluation.
* **Tech Stack:** Python, Streamlit, Scikit-Learn (TF-IDF Vectorization).

### Task 4: Object Detection and Tracking
* **Description:** A real-time computer vision system that captures webcam video feed, runs automated target detection, and maps persistent unique Tracking IDs using visual bounding boxes.
* **Tech Stack:** Python, OpenCV (Haar Cascade Classifiers).
